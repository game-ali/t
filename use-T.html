How to Use the Timetable

1. Adding a Course

To add a course to the timetable, follow these steps:

  1. Enter the Classroom Name in the provided input field.
  2. Select the Background Color for the classroom name header.
  3. Select the Day from the dropdown menu.
  4. Enter the Course Name and Course Code.
  5. Select the Start Time and End Time.
  6. Enter the Venue.
  7. Choose the Row Background Color.
  8. Click the Add Course button to add the course to the timetable.

2. Editing a Course

If you need to edit a course, follow these steps:

  1. Find the course you want to edit in the timetable.
  2. Click the Edit button next to the course.
  3. The course details will be populated into the input fields.
  4. Make the necessary changes to the course details.
  5. Click the Add Course button to save the changes.

3. Downloading the Timetable

To download the timetable as a PDF, follow these steps:

  1. Ensure that all courses have been added and edited as needed.
  2. Click the Download Timetable button.
  3. A PDF file of the timetable will be generated and downloaded to your device.

4. General Tips

Here are some general tips for using the timetable effectively:

  • Make sure to fill out all fields when adding or editing a course to avoid errors.
  • Use different colors for different courses to make the timetable more visually appealing and easier to read.
  • Double-check the start and end times to ensure they are accurate.
  • Save your work frequently by downloading the timetable PDF after making significant changes.
